Subtitle    : Changes in beta 2
Paragraph   :
{
   *  (AS)  The key to enable/disable the abbreviations is changed from ALT-A
            to CTRL-J. Now, users can enable or disable abbreviations too.
}

Paragraph   :
{
   *  (AS)  The abbreviations are now multi-language. Create an DUTCH.ABR
            file for the dutch abbreviations, GERMAN.ABR for germany, ... If
            the [LANGUAGE].ABR file is missing, TheEdit will use THEEDIT.ABR
}

Paragraph   :
{
   *  (AS)  The sysop and user can now press CTRL-E to list the abbreviations
            on the screen.
}

Subtitle    : WARNING!!!
Paragraph   :
{
   The language and help files are changed. The two new prompts for the
   abbreviations (title and error if no abbrevs found) are included. Please
   update your files!
}
